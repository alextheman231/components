This is a new patch release of the `@alextheman/components` package. It includes small, non-breaking changes and should require no refactoring. Please read the description of changes below.
<!-- alex-c-line-release-summary-end -->

## Description of Changes

<!-- user-editable-section-start -->
- Add a Storybook manager so that we can replace the Storybook logo with our package details.
    - We now show the package name and package version in the top-left corner of the page as a whole, replacing the initial Storybook logo.
    - It helps to make the site feel more professional and more independent from the default Storybook branding.
- Fix the Loader tests from making the docs page crash.
    - If you tried clicking on the Loader docs, an error would occur on the story due to Vitest not existing in the Storybook runtime.
    - This change isolates the tests that require direct Vitest imports in their own file, and also excludes them from the docs site itself.
    - This ensures that the Vitest-specific logic does not get rendered in the Storybook and hence does not cause any errors.
