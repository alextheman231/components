## Description of Changes

<!-- user-editable-section-start -->
- Make the button/tab display case-sensitively to how it was initially entered in the `ThemeProvider` defaults.
  - Previously the tab casing was all uppercase, which wasn't really an accurate reflection on how it was entered initially.
  - This changes that so that the casing is preserved in the display and therefore better reflects the input.
