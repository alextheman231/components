This is a new minor release of the `@alextheman/components` package. It introduces new features and other backwards-compatible changes that should require little to no refactoring. Please read the description of changes below.
<!-- alex-c-line-release-summary-end -->

## Description of Changes

<!-- user-editable-section-start -->
- Add a `loop` option to `AudioControls`
  - This will make the audio play on a loop if `true`.
  - Note that the looping behaviour is determined purely by the value of the prop. If you would like to give users the ability to determine if the audio should be looped or not, you will need to create a button or something like that to track this.
  - This will most likely be added more natively to this package in the near future.
